# Описание всех настроек и к чему они относятся

## Предисловие

Cписок настроек делится на три основные группы: **general**, **client**, **server**.

- **General** - команды применимые как для игры на сервере так и для игры в одиночку.

- **Client** - настройки специфичные только для одиночной игры.

- **Server** - настройки специфичные только для серверной стороны.

Есть так же настройки **colors** это список цветов в какие можно окрасить вывод команд.

## Список настроек

### General

- **home-description** - описание команды телепортации в точку дома

- **sethome-description** - описание команды смены точки дома

- **home-command** - написание команды телепортации домой

- **sethome-command** - написание команды смены точки дома

- **saves-path** - JSON файл сохранения данных игроков(данные хранятся по пути мира в папке data)

- **error-message** - сообщение которое показывается игроку при попытки телепорта если у него ещё нет точки дома

- **error-color** - цвет ошибки (берется из списка colors)

- **success-message** - сообщение которое появляется при смене точки дома

- **success-color** - цвет успешной смены (берется из списка colors)

- **readable-json** - данные игроков хранятся в JSON формате, этой настройкой можно изменить будут ли они хранится в человекочитаемом виде или нет.

- **home-logs** - будут ли вестись логи для телепортации домой (если установлен мод на логи)

- **sethome-logs** - будут ли вестись логи для смены точки дома (если установлен мод на логи)

### Client

- **is-home-cheat** - является ли команда телепортации домой читерской

- **is-sethome-cheat** - является ли команда смены точки дома читерской

### Server

- **can-use-home-unauthorized** - может ли использовать команду телепортации домой неавторизованный игрок

- **can-use-sethome-unauthorized** - может ли использовать команду смены дома неавторизованный игрок

- **home-admin-logs** - будут ли вестить логи об действиях админа с командой телепортации домой

- **sethome-admin-logs** - будут ли вестись логи об действиях админа с командой смены точки дома
